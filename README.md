# AIORCH Demo Projects

Sample tasks for testing [AIORCH](https://aiorch.ai) — AI code orchestration platform.

## Projects

| Project | Type | Agents | Time | Description |
|---------|------|--------|------|-------------|
| [bookmark-api](bookmark-api/) | Session | 3 | 5-8 min | Bookmark manager with collections, tags, search, import/export |
| [invoice-management-api](invoice-management-api/) | Pipeline | 4 | 10-15 min | Multi-tenant invoice system with security and tests |

## Quick Start

1. Install AIORCH: `curl -fsSL https://aiorch.ai/install.sh | bash`
2. Clone this repo on your server: `git clone https://github.com/aiorch-ai/demo-projects.git`
3. Open the AIORCH dashboard and create a session or pipeline
4. Point project root to the cloned directory
5. Paste the prompt from the project's `prompts/` folder

See each project's README for detailed setup instructions.
